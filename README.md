# 리액트라우터를 이용한 라우팅처리

## 개발환경 / 실행방법
### `Node.js version`
16.19.0

### `yarn start`
yarn start 커맨드를 입력하면 local host 포트3000 에서 실행됩니다

### `yarn build`
yarn build 커맨드를 입력하면 빌드를 진행합니다
